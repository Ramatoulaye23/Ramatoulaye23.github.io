    Classic Model->template eCommerce

    L'objectif de ce projet est la creation d'un DASHBOARD d'une administration La realisation de ce projet m'a été d'une grande utilité étant donner que j'ai pu mettre en pratique les connaissances que j'ai acquise lors de ma reconversion. Différente étapes de realisation:

    • Modelisation de mes différentes tables a travers la méthode merise (MCD,MLD)avec l'outils figma J'ai commencer par créer mon MCD: modele conceptuel des données en mettant les clef primaires ainsi que les cardinalités. Puis de ce modèle j'ai générer le MPD : modèle physique des données: dans lequel j'ai fais apparaitre les clef secondaires ainsi que les relations qui se trouvaient entre mes différente tables
    • Import d'une base de données existante dans PhpMyAdmin La base de données ayant été importé correctement, a travers l'onglet concepteur j'ai afficher mes tables puis j'ai appliqué mon MPD sur mes tables en créant les différentes relation entre les tables et en mettant les clef secondaire sur les tables concerné
    • Préparation de mes requetes SQL: j'ai crée un fichier SQL en illustrant les différents cas de figure pour afficher les données de mes tables a travers des jointures de tables.
    • PDO: Mise en relation de mes requetes preparé en les injectant dans mon projet a travers un PDO()
    • Application de la méthode MVC
